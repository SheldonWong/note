# 004 运算符



## 01 算数运算符

+、-、*、/、%（取模/取余）、++、--

> 1. 两个int型的整数相除（/），即使结果是小数，也会输出int型整数（规定）。
> 2. 两个小数不能进行取模运算，但可以进行除法运算。

在JavaScript中1/3*3 = 0.9999

但在C++语言中，结果却等于1

```C++
	double a = 1;
	double b = 3;
	double c = a/b;
	double res = c*3;
	cout << res << endl; // 1
```
