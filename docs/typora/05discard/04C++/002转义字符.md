# 002 转义字符

| 转义字符  | 含义                           | ASCII码值（十进制） |
| --------- | ------------------------------ | ------------------- |
| \a        | 警报                           | 007                 |
| \b        | 退格，将当前位置移到前一列     | 008                 |
| \f        | 换页，将当前位置移到下页开头   | 012                 |
| **\n**    | 换行，将当前位置移到下一行开头 | 010                 |
| \r        | 回车，将当前位置移到本行开头   | 013                 |
| **\t**    | 水平制表                       | 009                 |
| \v        | 垂直制表                       | 011                 |
| **\\ \\** | 代表一个反斜杠                 | 092                 |
| \\'       | 代表一个单引号                 | 039                 |
| \\"       | 代表一个双引号                 | 034                 |
| \?        | 代表一个问号                   | 063                 |
|           |                                |                     |
