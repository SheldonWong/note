```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>client家族</title>
</head>
<body>
<script>
    /*
    clientWidth与clientHeight
        ·网页可见区域的宽document.body.clientWidth
        ·网页可见区域的高document.body.clientHeight
    clientLeft和clientTop(没有讲)
        ·返回的是元素边框的borderWidth
        ·如果不指定一个边框或者不定位该元素，其值就为零
     */
    /*
    三大系列区别
    offsetWidth,offsetHeight是【内容的宽（高） + padding + border】
    clientWidth,clientHeight是【内容的宽（高） + padding】（内容超出盒子高度不增加）
    scrollWidth,scrollHeight是【内容的宽（高） + padding】（内容超出盒子高度增加）
     */
</script>
</body>
</html>
```
