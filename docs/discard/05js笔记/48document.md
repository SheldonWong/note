    /*
    document.head()获取头部
    document.body()获取身体
    document.title()获取标题
    document.documentElement()获取整个HTML

    BackCompat：标准兼容模式关闭
        浏览宽度：document.body.clientWidth;
    Css1Compat：标准兼容模式开启
        浏览宽度：document.documentElement.clientWidth;

    故事背景：早期的浏览器Netscape 4和Explorer 4对css进行解析时，并未遵守w3c标准，这时解析方式被称为quirks mode怪异模式，随着W3C的越来越重要，浏览器依照W3C解析称strict mode为严格模式
    关键词：Document.compatMode;
    使用细节：见04进阶小项目>scroll家族>02json封装
     */
</script>
</body>
</html>
